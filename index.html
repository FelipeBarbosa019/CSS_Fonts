Fontes:


Existem cinco famílias de fontes no CSS, sendo elas:

  • Serif fonts: têm um pequeno traço nas bordas de cada letra. Eles criam uma sensação de formalidade e elegância.

    Exemplo: Este é um texto escrito em família de fontes Serif.
    Sintaxe: font-family: 'Source Serif Pro', serif;


  • Sans-serif: têm linhas limpas (sem pequenos traços anexados). Eles criam um visual moderno e minimalista.

    Exemplo: Este é um texto escrito em família de fontes Sans-serif.
    Sintaxe: font-family: 'Roboto', sans-serif;


  • Monospace aqui todas as letras têm a mesma largura fixa. Eles criam uma aparência mecânica.

    Exemplo: Este é um texto escrito em família de fontes Monospace.
    Sintaxe: font-family: 'Inconsolata', monospace;


  • Cursive: imitam a caligrafia humana.

    Exemplo: Este é um texto escrito em família de fontes Cursive.
    Sintaxe: font-family: 'Zen Loop', cursive;


  • Fantasy: são fontes decorativas/lúdicas.

    Exemplo: Este é um texto escrito em família de fontes Fantasy.
    Sintaxe: font-family: 'papyrus', fantasy;


Manipulação de texto:

O comando writing-mode define se as linhas do texto são dispostas horizontalmente ou verticalmente. Além disso, estabelece a direção que os blocos seguem. Exemplos:


Valor vertical-lr: Texto na vertical escrito da esquerda para direita.

Valor vertical-rl: Texto na vertical escrito da esquerda para direita.

Valor horizontal-tb: Conteúdo é disposto horizontalmente da esquerda para a direita, verticalmente de cima para baixo.
